# Movie Management

Endpoints for managing movies and synchronizing with TheMoviesDB.

## Sync Movies

Synchronize multiple movies from TheMoviesDB with the local database.

### Request

```bash
POST /api/movies/sync
```

### Request Body

<ParamField body="SyncMoviesRequest" type="object">
  <Expandable title="Properties">
    <ParamField body="movieIds" type="array" required>
      Array of TheMoviesDB movie IDs to sync
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="200: OK" type="object">
  Returns the sync operation results
</ResponseField>

## Sync Single Movie

Synchronize a single movie from TheMoviesDB.

```bash
GET /api/movies/sync/{id}
```

### Path Parameters

<ParamField path="id" type="string" required>
  TheMoviesDB movie ID
</ParamField>

### Response

<ResponseField name="200: OK" type="object">
  Returns the synchronized movie details
</ResponseField>

## Get Movie by ID

GET /api/movies/{id}

### Path Parameters

<ParamField path="id" type="integer" required>

</ParamField>

### Response

<ResponseField name="200: OK" type="object">

</ResponseField>
