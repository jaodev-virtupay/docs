# Two-Factor Authentication

Manage two-factor authentication (2FA) for enhanced security.

## Generate 2FA Code

Request a new two-factor authentication code.

### Request

```bash
GET /api/authenticate/twofactor
```

### Response

<ResponseField name="secretKey" type="string">
  The secret key for 2FA setup
</ResponseField>

<ResponseField name="qrCodeUrl" type="string">
  URL for the QR code to scan with authenticator app
</ResponseField>

```json
{
  "secretKey": "JBSWY3DPEHPK3PXP",
  "qrCodeUrl": "otpauth://totp/YourApp:user@example.com?secret=JBSWY3DPEHPK3PXP&issuer=YourApp"
}
```

## Enable 2FA

Validate and enable two-factor authentication for the account.

### Request

<ParamField body="code" type="string" required>
  The verification code from the authenticator app
</ParamField>

```bash
POST /api/authenticate/twofactor
Content-Type: application/json

{
  "code": "123456"
}
```

### Response

<ResponseField name="success" type="boolean">
  Indicates if 2FA was enabled successfully
</ResponseField>

<ResponseField name="message" type="string">
  Status message about the 2FA setup
</ResponseField>

```json
{
  "success": true,
  "message": "Two-factor authentication enabled successfully"
}
```

<ResponseExample>
```json
{
  "success": true,
  "message": "Two-factor authentication enabled successfully"
}
```
</ResponseExample>
