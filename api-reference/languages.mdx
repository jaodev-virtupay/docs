# Language Management

Endpoints for managing movie languages in the system.

## List Languages

Retrieve a paginated list of languages.

### Request

```bash
GET /api/languages
```

### Query Parameters

<ParamField query="languageName" type="string">
  Filter languages by name
</ParamField>

<ParamField query="pageNumber" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="pageSize" type="integer" default="20">
  Number of items per page
</ParamField>

### Response

<ResponseField name="200: OK" type="object">
  Returns a paginated list of languages
</ResponseField>

## Get Language by ID

Retrieve a specific language by its ID.

```bash
GET /api/languages/{id}
```

### Path Parameters

<ParamField path="id" type="integer" required>
  The ID of the language
</ParamField>

### Response

<ResponseField name="200: OK" type="object">
  Returns the requested language
</ResponseField>
